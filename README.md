# ekeynox

A typing test app. Pick a mode, type, get stats.

## Modes

- **Speed** — type a set number of words as fast as you can
- **Timer** — type as much as you can before time runs out

Supports words (top 1k) and quotes, in English and French. AZERTY/QWERTY layouts with an optional on-screen keyboard.

## Setup

```bash
pnpm install
pnpm --filter front dev
```

## Stack

SolidJS, SolidStart, TypeScript, solid-styled, anime.js, chart.js
